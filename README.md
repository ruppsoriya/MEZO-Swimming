# SWIM X Club Website

Static website for swimming programs, partnerships, booking, finance planning, and brochure sharing.

## Open

Open `index.html` in a browser.

## Pages

- `index.html` - main website
- `brochure.html` - printable partner brochure

## Features

- Package cards with detailed inclusions and suggested prices
- Package comparison table
- Package pricing calculator
- Finance calculator for profit estimation
- Booking form and admin schedule editor

## Use

1. Open `index.html`.
2. Select a package in Programs to see session plans.
3. Use the pricing calculator to estimate package revenue.
4. Open `brochure.html` and print or save as PDF for outreach.
