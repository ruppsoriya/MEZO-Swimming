## 🔧 How to Customize

### Change Salaries
Edit in `script.js`:
```javascript
{
  id: 'coach_001',
  name: 'Coach Maria',
  employment: 'Full-Time',
  monthlySalary: 650,  // Change this
  hourlyRate: 15,
  maxHoursPerWeek: 40
}
```

### Add New Team Members
Copy a template and add to `teamMembers[]`:
```javascript
{
  id: 'staff_006',
  name: 'New Person Name',
  role: 'Position Title',
  employment: 'Full-Time', // or 'Part-Time'
  email: 'email@swimxclub.com',
  phone: '+855 123 xxxx',
  certifications: ['Cert1', 'Cert2'],
  specialties: ['Specialty1', 'Specialty2'],
  monthlySalary: 500, // for full-time only
  hourlyRate: 0, // or rate for part-time
  maxHoursPerWeek: 40,
  active: true,
  joinDate: '2024-05-05'
}
```

### Adjust Payroll Calculations
The payroll summary automatically calculates:
- Full-time total: Sums all `monthlySalary` values
- Part-time total: Estimates `hourlyRate × maxHoursPerWeek × 4.3`
- Grand total: Adds both together
